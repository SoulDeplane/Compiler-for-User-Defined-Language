# Nova Programming Language

Nova is a simple statically-typed programming language
implemented using Flex, Bison, and C.

## Features
- Variables (int, string)
- Arithmetic expressions
- if / else
- for loops
- String printing
- Compile-time optimization
- 8086 assembly code generation

let a = 2 + 3
print a

if a > 3 [
    print "Greater"
]
